## Installation

Follow these steps to set up the project locally:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/yashwaghmare/nova.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd nova
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```
